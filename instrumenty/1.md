(*Это переменные, которые являются членами класса. Как только для **final** поля класса присвоено значение, оно не может быть изменено. **Эти переменные должны быть инициализированы либо в момент их объявления, либо в конструкторе класса**. Если **final** поле не инициализировано напрямую при объявлении, то оно должно быть инициализировано в каждом конструкторе класса. Если класс имеет несколько конструкторов, каждый из них должен инициализировать **final** поле, чтобы гарантировать, что перед использованием поле будет иметь значение.*)
